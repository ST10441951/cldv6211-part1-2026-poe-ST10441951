# EventEase - Event Management System

## Project Overview
A professional ASP.NET Core MVC application designed to manage university venues, schedule events, and track bookings.

## Features
* **Venue Management:** Full CRUD functionality with image support.
* **Event Scheduling:** Linked to bookings for seamless organization.
* **Booking System:** Relationship-based tracking of dates and statuses.
* **Responsive Dashboard:** Built with Bootstrap 5.

## Tech Stack
* **Backend:** ASP.NET Core 10.0 (MVC)
* **Database:** Entity Framework Core (SQL Server)
* **Frontend:** Bootstrap 5, CSS3, HTML5
